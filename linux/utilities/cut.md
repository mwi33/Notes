# cut
Cut is a command line utility that can be used to cut sections out of a file or piped data to a standard output.  The command cuts parts of a line by field, delimeter, byte position and character.

## General syntax
~~~ bash

cut [option] [file]

~~~

## Options
~~~ bash

-f field
-b bytes
-c character
-d delimeter
--complement
-s only delimited
--output-delimeter


~~~
